# Firewall & IDS Project

This project implements:
- Firewall using iptables
- Intrusion Detection using Snort

## Tools Used:
- Kali Linux
- iptables
- Snort

## Features:
- Blocks unauthorized traffic
- Detects attacks in real time
- Generates alert logs
